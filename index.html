Content

This unit introduces students to professional practice in IT. Students will learn version control using Git, collaborative workflows, teamwork principles, and industry-standard development practices.

The unit focuses on developing communication skills, project management understanding, and practical GitHub usage required in real-world software development environments.

Hurdle Requirement

To pass this unit, students must achieve at least 50% overall and complete all required assessment components.
